# Файли курсу

Помістіть у цю папку Excel-файли для уроків. Очікувані імена:

- `lesson-XX-practice.xlsx` — файл для практики до уроку XX
- `lesson-XX-validation.xlsx` — файл для самоперевірки до уроку XX

Де `XX` — номер уроку з провідним нулем (наприклад, `lesson-02-practice.xlsx`).

Сторінка `lessons.html` уже містить посилання на ці файли — як тільки ви додасте їх у цю папку, кнопки завантаження запрацюють автоматично.
